# Icy Silvertail portfolio

This repository is to track changes of my Newcities site that shows my Art portfolio. My artist pseudonym is **Icy Silvertail**

[Link to the site](https://icy-silvertail.neocities.org/)

Pictures for the portfolio itself are not uploaded here. I am thinking on using a backend for the pictures but I still have not decided.

**To run this site in your local environment, you just need to install Pyhton and then use Pip to install some libraries. I used Pyhton 3.14.3**

## Obtain gallery.json

You need to use the Python script `obtain_json_images.py` to obtain the file. I recommend you to track it on your git repository.
